# Calculator Neumorphism

A clean and modern **neumorphism-style calculator** built using **HTML, CSS, and JavaScript**.  
This calculator offers a sleek and soft UI with smooth button interactions. It is fully responsive, works on both desktop and mobile devices, and supports basic arithmetic operations like addition, subtraction, multiplication, and division.  

It is easy to customize and perfect for learning modern frontend design with a focus on UI/UX and neumorphic styling.
